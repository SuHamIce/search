# Java 多线程文件检索项目

## 项目介绍
这是一个基于 Java 开发的文件关键字搜索工具，通过多线程并发处理提高检索效率，属于个人学习与课程作业项目。

## 功能
- 支持本地文本文件关键字检索
- 使用线程池实现多线程处理
- 控制台交互，支持输入关键词搜索
- 支持退出指令
- 支持精准搜索和模糊搜索
- 支持多文件检索

## 技术栈
- Java
- 多线程 / ThreadPoolExecutor
- 文件 IO
- IntelliJ IDEA

## 使用方法
1. 运行主类启动程序
2. 输入需要搜索的关键字
3. 查看检索结果
4. 输入退出指令结束程序
